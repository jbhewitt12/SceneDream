# SceneDream Frontend

React + TypeScript dashboard for document status, pipeline launch, settings, and generated image workflows.

## Prerequisites

- Node.js 18+
- npm

## Local Development

```bash
cd frontend
npm install
npm run dev
```

Default local URL: http://localhost:5173

## Frontend Environment

`frontend/.env` controls frontend-only variables. Defaults in this repository point to a local backend:

- `VITE_API_URL=http://localhost:8000`

Update this value if your backend runs elsewhere.

## Lint and Build

```bash
cd frontend
npm run lint
npm run build
```

## Generate API Client

Preferred from repository root:

```bash
./scripts/generate-client.sh
```

Manual fallback:

```bash
cd backend
uv run python -c "import json; from app.main import app; from pathlib import Path; Path('../openapi.json').write_text(json.dumps(app.openapi(), indent=2))"
cd ../frontend
npm run generate-client
```

## Route Tree Regeneration

`src/routeTree.gen.ts` is generated by TanStack Router tooling. Regenerate it by running a build:

```bash
cd frontend
npm run build
```

## Frontend Structure

- `src/routes/`: route modules
- `src/components/`: reusable UI components
- `src/api/`: API helpers and domain adapters
- `src/client/`: generated OpenAPI client
- `src/hooks/`: custom hooks
- `src/theme/`: Chakra UI theme and tokens
