# SceneDream Frontend

The frontend uses Vite, React, TypeScript, Chakra UI, TanStack Router, and a generated OpenAPI client.

## Local Development

```bash
cd frontend
npm install
npm run dev
```

## Lint and Build

```bash
cd frontend
npm run lint
npm run build
```

## Generate API Client

Preferred from repo root:

```bash
./scripts/generate-client.sh
```

Manual fallback:

```bash
cd backend
uv run python -c "import json; from app.main import app; from pathlib import Path; Path('../openapi.json').write_text(json.dumps(app.openapi(), indent=2))"
cd ../frontend
npm run generate-client
```

## Route Tree Regeneration

`routeTree.gen.ts` is generated by TanStack Router during frontend tooling/build. If routes change, run:

```bash
cd frontend
npm run build
```

## Frontend Structure

- `src/routes/`: route modules
- `src/components/`: reusable UI components
- `src/client/`: generated API client
- `src/hooks/`: custom hooks
- `src/theme.ts`: Chakra theme
